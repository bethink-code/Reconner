Everything anyone needs to produce on-brand work for lekana — the mark, the colours, the type, the words, the rules. Nothing more. If it's not in here, it isn't part of the brand yet.

1. The mark

Three primitive shapes. Dot, line, dot.

A vertical line between two dots. Pure filled shapes only — no borders, no strokes, no effects. The three elements are equally spaced.

Dark on sunshine — primary
Cream on dark — closing card, footer

Rules

  • Minimum size16px height. Below this, use wordmark only.
  • ContrastThe sunshine yellow mark must never appear on a light background. Fails contrast every time.
  • ForbiddenDon't stretch, rotate, recolour, or add effects. Don't outline the dots. Don't tilt the line.
  • SVG sourceviewBox="0 0 80 40". Circles r=6 at cx=10 and cx=70, cy=20. Rect x=38 y=6 w=4 h=28 rx=2.

2. Wordmark and lockup

lekana

Rules

  • Always lowercaselekana. Never Lekana. Never LEKANA.
  • FontNunito 400. Never bold. The lightness is what makes it sit calm.
  • Letter-spacing-0.01em. Slightly tight.
  • Lockup spacingMark and wordmark separated by approximately the width of one wordmark letter.
  • Mark heightMark cap height ≈ wordmark cap height. They line up at the optical centre.

Primary: dark fill, cream text. Secondary: cream fill, dark text. Never on sunshine yellow. Orange reserved exclusively for the Send request button.

Layout principles

  • No borders on cardsSeparation comes from background colour, not from outlines.
  • No drop shadowsCards rest on the page, they don't float above it.
  • Generous whitespaceCards have substantial internal padding. Sections breathe.
  • Centred hero, left-aligned dataMarketing surfaces centre everything. The application aligns everything left.
  • Stepper bandsReconciliation flow uses coloured bands (plum, deep teal, burnt orange, sunshine) to mark stages. Used only inside the flow, not elsewhere.

9. Using this brief

For Claude or any LLM

Paste the relevant section into the prompt, or attach this whole file. Reference colours by hex, fonts by name and weight, and the locked strings by their exact wording. The mark's SVG source is in section 1 — copy it verbatim.

For a designer or video editor

Read sections 1–4 once. Print the colour and type pages if it helps. The locked strings in section 6 are the ones that get repeated across surfaces — they should never be rewritten by anyone making creative.

For Garth or Pieter

If something feels off in a piece of creative, check it against this document first. Either the work is wrong, or the brief needs updating. Don't tolerate drift in either direction.
